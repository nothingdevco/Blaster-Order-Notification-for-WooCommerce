== Description ==

Blaster.my WooNotification is a WordPress plugin that allows you to send WhatsApp notifications to customers when their orders are completed or being processed in WooCommerce. This plugin is designed exclusively for app.blaster.my subscribers.

Main features:
- Send WhatsApp notifications to customers upon order completion or processing.
- Customizable notification messages using template tags.
- Support for variables like customer name, order ID, order details, and shipping address.
- Easy-to-use settings page to configure the API key and notification messages.

== Installation ==

1. Upload the `blaster-my-woonotification` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to "Settings" > "Blaster.my Settings" to configure the plugin.

== Frequently Asked Questions ==

= What is the API key? =
The API key is required to connect the plugin with app.blaster.my's WhatsApp notification service. To obtain the API key, you need to be a subscriber of app.blaster.my and follow the instructions provided by the service.

= How can I customize the notification messages? =
You can customize the notification messages on the plugin settings page. The plugin supports template tags that can be used to include dynamic information like customer name, order ID, order details, and shipping address in the messages.
